We provides two installation methods for you to choose from.

## Quick Installation with CLI

Use our CLI tool to quickly install the required components:

```bash
npx shadcn@latest add <component-name>
```

For example, install the button component:

```bash
npx shadcn@latest add button
```

CLI will automatically complete the installation and configuration process, without any additional steps.

## Manual Installation

If you prefer to install manually, please follow the steps below:

<Steps>

<Step>

Follow the [Shadcn UI installation guide](https://ui.shadcn.com/docs/installation) to install the core packages.

</Step>

<Step>

Install the following packages:

```bash
npm install motion lucide-react
```

</Step>

<Step>

## Let's get started

Now that you've installed the core packages and the required components, you're ready to start building with AnnUI!

```tsx title="app/page.tsx"
import { Button } from "@/components/ui/button"

export default function Home() {
  return <Button>Click me</Button>
}
```
